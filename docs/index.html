Layer-3 UDP Tunnel / Pure Zig / Static Binary

Subnetra

Wrap raw Layer-3 IP packets into a private UDP tunnel — a tiny static single binary that builds a hub-and-spoke virtual subnet across Linux hosts and lightweight containers.

< 512KB Zero third-party dependencies, fully static musl build; released for amd64, arm64, armv7 and armv5.
Single Hub Relay Core
01 / Ingress

TUN Ingress

Read raw IPv4 packets from the virtual L3 device.

02 / Shield

Encrypt & Seal

ChaCha20-Poly1305 over a 20B private header; silent drop on failure.

03 / Hub

Star Relay

Hub relays by policy to each spoke; never back to its source.

04 / Policy

Policy Routing

Longest-prefix CIDR match, hot-swapped via RCU — no restart.

05 / Egress

Spoke Egress

Verify epoch, nonce, anti-replay and inner source, then deliver.

0

Data-plane allocations

Reactor and crypto paths lock buffers after startup — low jitter, steady memory.

1

Single-threaded epoll reactor

Edge-triggered loop: lock-free, no data races — ideal for small devices.

64B

Fixed per-packet wire overhead

20B header + 16B AEAD tag + 28B outer IPv4/UDP — sizes a safe MTU.

PSK

Per-link independent key

One PSK per peer link; directional keys, epoch and nonce prevent reuse.
